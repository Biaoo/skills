# Agent Behavior Skills

Cross-domain skills that encode durable preferences for how an Agent should approach work.

- [calibrated-assertiveness](./calibrated-assertiveness/SKILL.md) - preserve clear intent without defensive caveats, scope reduction, or weakened verification.
- [outcome-first-implementation](./outcome-first-implementation/SKILL.md) - prioritize requested outcomes over implementer convenience when AI owns the implementation.
